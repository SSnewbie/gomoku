# gomoku
五子棋联机版

实现一个五子棋的游戏，可以联机对战，也可以单机与AI对战

## 如何使用
```
git clone https://github.com/SSnewbie/gomoku.git
npm install
npm run bulid
npm run server
```

## 技术
- `webpack` + `express`+ `socket.io` + `es6`
